### СТРУКТУРА И КОНТЕНТ ГЛАВНОЙ СТРАНИЦЫ:

**Хедер и основная навигация:**
- [x] Логотип Wattpad отображается в левом верхнем углу
- [x] Логотип кликабелен и ведет на главную страницу при любом расположении
- [x] Поле поиска присутствует в хедере, доступно для ввода
- [x] Иконка поиска (лупа) кликабельна и активирует поле ввода
- [x] Кнопка "Community" отображается и ведет в разделы сообщества
- [x] Раздел "Browse" присутствует и открывает выпадающее меню жанров
- [x] Кнопка "Write" отображается и открывает меню создания новой истории
- [x] Для авторизованного пользователя отображается аватар/иконка профиля
- [x] Для неавторизованного пользователя отображаются кнопки "Login" и "Sign Up"
- [x] Кнопка "Try Premium" отображается в видимом месте

**Контентные блоки:**
- [x] Лента персонализированных рекомендаций отображается для авторизованного пользователя
- [x] Популярные/трендовые книги показываются для неавторизованного пользователя
- [x] Категории/жанры отображаются с обложками и названиями
- [x] Персонализированные рекомендации основаны на истории чтения

### ФУНКЦИОНАЛЬНОСТЬ ЭЛЕМЕНТОВ:
- [x] Клик по аватару пользователя открывает выпадающее меню
- [x] Меню содержит: My Profile, Inbox, Notifications, Library, Language settings, Help, Settings, Log out
- [x] Все пункты меню ведут на соответствующие страницы
- [ ] Клик на log out > открывается подтверждающее окно
- [x] Меню закрывается при клике вне области
- [x] Карточки книг кликабельны и ведут на страницу книги

### ВЕБ-СПЕЦИФИЧНЫЕ ПРОВЕРКИ:

- [x] Размеры элементов адаптируются к разным разрешениям экрана
- [x] Клавиатурная навигация работает (Tab, Enter)
- [x] Плавный скролл между секциями
- [ ] Кнопка "Back to Top" появляется при скролле вниз

### МОБИЛЬНАЯ ВЕРСИЯ:

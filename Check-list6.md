### Общие:

- [x] Сообщение при проблемах с загрузкой библиотеки

 ### Раздел Current read(аналогично для Archive):
 - [x] Все истории и их обложки отображаются в разделе
 - [x] При наведении курсора на историю > открывается меню: start reading/continue reading, details, archive, add to list, элемент корзины(delete)
 - [x] "Start Reading" открывает первую главу
 - [x] "Continue Reading" открывает последнюю прочитанную главу
 - [x] "Details" переходит на страницу книги
 - [x] "Add to List" открывает выбор списков для добавления
 - [x] "Archive" перемещает историю в архив
 - [x] "Delete" удаляет историю с подтверждением
 - [x] Все элементы меню кликабельные и выделяются при наведении на них курсора 
 - [x] Под историями отображается шкала прогресса чтения
 - [x] Под каждой историей отображается иконка и ник автора, количество просмотров и голосов у книги
 - [x] Историю можно добавить в список чтения(add to list) > появляется в разделе reading list

---

### Раздел Reading list:
- [x] Возможность создать список для чтения при клике на кнопку "Create reading list"
- [x] Можно ввести 1 - 98 символов в поле названия списка для чтения 
- [ ] Только пробелы в поле > кнопка create не активна
- [x] Поле принимает цифры, символы, эмодзи, иероглиыв, лат. и кирл.
- [ ] Информативное сообщение при превышения лимита символов в поле названия списка для чтения
- [x] Отображается приватный раздел понравившихся списков для чтения и их количество
- [x] Переход в понравившиеся списки > отображается список коллекций
- [ ] Нет понравившихся списков > сообщение об отсутвии и подсказки как добавить списки
- [x] На главной страницы раздела отображается пользовательский список для чтения
- [x] Есть возможность перетащить коллекцию для изменения порядка в списке с помощью иконки для перетаскивания
- [x] Перетаскивание реализуется через долгое нажатие > появляется курсор "рука", перетаскивание плавное 
- [x] При клике на троеточие открывается контекстное меню(редактировать, удалить, поделиться)
- [ ] Все элементы меню имеют hover эффекты
- [x] При нажатии на пользовательский список > открывается страница со списком историй
- [x] На странице присутсвуют: отображение количества историй, редактирование, удаление, кнопка лайка(функция добавить в понравившиеся), иконки соцсетей(функция поделиться)
- [ ] Все иконки с hover эффектами
- [x] У каждой истории присутствует: обложка, имя автора, описание истории, количество просмотров/голосов/глав,теги
- [x] Нажатие на историю > переход на главную страницу книги

### Базовые проверки:
- [ ] 1. На главной странице отображаются кнопки "Зарегистрироваться", "Войти", "Скачать приложение".
- [ ] 2. Нажатие на кнопку регистрации > открывает модальное окно с выбором способа регистрации.
- [ ] 3. Окно регистрации содержит: регистрацию через google, facebook и email, ссылку "У меня уже есть аккаунт".
- [ ] 4. Все кнопки и интерактивные элементы активны и кликабельны, имеют визуальный feedback при наведении.
- [ ] 5. Закрытие модального окна крестиком или кликом вне области возвращает на главную страницу.
- [ ] 6. Переход по ссылке "У меня уже есть аккаунт" открывает окно авторизации

---

### Регистрация через Google:
- [ ] 1. Нажатие "Продолжить с Google" открывает стандартное окно выбора аккаунта Google
- [ ] 2. Пользователь может выбрать аккаунт > переход на страницу регистрации.
- [ ] 3. Регистрация с уже существующим аккаунтом Google и аккаунтом Wattpad > автоматический вход в систему.
- [ ] 4. Регистрация с новым аккаунтом Google > успешное создание аккаунта Wattpad.
- [ ] 5. При отмене авторизации в Google пользователь остается на странице Wattpad.

 **Поле username:**
  - [ ] 1. Поле активное и обязательное.
  - [ ] 2. Ввод 6-20 символов > успешно.
  - [ ] 3. Ввод менее 6 символов > сообщение об ошибке.
  - [ ] 4. Ввод более 20 символов > сообщение об ошибке.
  - [ ] 5. Ввод пробелов > сообщение об ошибке.
  - [ ] 6. Ввод кириллицы > сообщение об ошибке.
  - [ ] 7. Ввод спецсимволов кроме нижнего подчеркивания > сообщение об ошибке.
  - [ ] 8. Ввод уже существующего username > сообщение об ошибке.

**Поле дата рождения:**
  - [ ] 1. Пользователь может выбрать дату рождения (поле является обязательным).
  - [ ] 2. Выбранные даты отображаются в полях месяц/день/год.
  - [ ] 3. Возраст старше 13 > успешно.
  - [ ] 4. Возраст младше 13 лет > сообщение об ошибке, невозможно зарегистрироваться.
  - [ ] 5. Дата из будущего > сообщение об ошибке.
  - [ ] 6. Дата рождения более 120 лет назад > сообщение об ошибке.
  
**Поле "существительное" (Pronoun):**
  - [ ] 1. Поле необязательное.
  - [ ] 2. При клике открывается выпадающий список с вариантами обращений.
  - [ ] 3. Выбранное обращение отображается в поле.
  - [ ] 4. Возможность очистить выбранное значение.
  
**Поле пароль:**
  - [ ] 1. Поле не обязательное.
  - [ ] 2. ввод 10-20 символов, цифр, спецсимвол, строчные и заглавные латинские буквы > успешно.
  - [ ] 3. Пустое поле > успешно.
  - [ ] 4. Только буквы > сообщение об ошибке.
  - [ ] 5. Только цифры > сообщение об ошибке.
  - [ ] 6. Ввод пробелов > сообщение об ошибке.
  - [ ] 7. Пароль содержит имя пользователя > сообщение об ошибке.
  - [ ] 8. Ввод 9 символов > сообщение об ошибке.
  - [ ] 9. Ввод 10 символов > успешно.
  - [ ] 10. Ввод 15 символов > успешно.
  - [ ] 11. Ввод 20 символа > успешно.
  - [ ] 12. Ввод 21 символа > сообщение об ошибке.
  - [ ] 13. Ввод кириллицы > сообщение об ошибке.
  - [ ] 14. Кнопка показать/скрыть пароль активна.
  - [ ] 15. При включенном "показать пароль" отображается точный введенный текст.

### Регистрация через email:
- [ ] 1. Нажатие "Зарегистрироваться через email" открывает форму с полями: email, дата рождения, username, пароль, повторный пароль
- [ ] 2. С валидным email, датой рождения, username, паролем > успешно, отправляется письмо с подтверждением.
- [ ] 2. Поле пароль обязательное.
- [ ] 3. Регистрация с уже существующим email > сообщение об ошибке.
- [ ] 4. Регистрация с невалидным форматом email > сообщение об ошибке.
- [ ] 5. Поле email обязательно для заполнения.
- [ ] 6. Поле пароль обязательно для заполнения.

**Поле email:**
- [ ] 1. Валидный email формата user@domain.com > успешно.
- [ ] 2. Email с поддоменом user@sub.domain.com > успешно.
- [ ] 3. Email без @ > сообщение об ошибке.
- [ ] 4. Email без домена > сообщение об ошибке.
- [ ] 5. Email с пробелами > сообщение об ошибкe.
- [ ] 6. Длинный email (более 254 символов) > сообщение об ошибке.
- [ ] 7. Email с кириллицей > сообщение об ошибке.
- [ ] 8. Пустое поле > сообщение об ошибке.

**Поле пароль:**
- [ ] 1. **[Комментарий: Все проверки пароля аналошичны в Google]**.
- [ ] 2. Поле обязательно для заполнения.

 **Поле повторный пароль:**
  - [ ] 1. Ввод пароля, совпадающего с первым полем > успешно.
  - [ ] 2. Ввод пароля, не совпадающего с первым полем > сообщение об ошибке.
  - [ ] 3. Пустое поле > сообщение об ошибке.
  - [ ] 4. Кнопка показать/скрыть пароль работает для обоих полей независимо.

**Чек-бокс "Условия обслуживания и Политика конфиденциальности":**
  - [ ] 1. Не устанавливать флажок и нажать "Зарегистрироваться" > сообщение об ошибке.
  - [ ] 2. Установить флажок и нажать "Зарегистрироваться" >  успешно.
  - [ ] 3. Нажать на ссылки "Условия обслуживания" и "Политика конфиденциальности" > успешно открываются в новых вкладках.
  - [ ] 4. Страницы условий и политики доступны и содержат актуальный контент
 
**Чек-бокс "Маркетинговые письма":**
  - [ ] 1. Оставить чек-бокс пустым и успешно зарегистрироваться > успешно.
  - [ ] 2. Установить флажок и успешно зарегистрироваться > успешно, подписка на рассылку подтверждена.
